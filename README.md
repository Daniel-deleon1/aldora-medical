## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Daniel-deleon1/aldora-medical.git
   ```
2. Navigate to the project folder:
   ```bash
   cd aldora-medical
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## How to Deploy to Netlify
1. Create a Netlify account if you don't have one already.
2. Go to the Netlify dashboard and click on "New site from Git."
3. Connect your GitHub account and select this repository.
4. Follow the instructions to deploy your site. Make sure the build command is `npm run build` and the publish directory is `dist/`.
